# App Resources

## Icon

Place your app icon here as `photorec-refinery.png` (512x512 or larger recommended).

Briefcase will automatically generate platform-specific icons:
- macOS: `.icns` file
- Windows: `.ico` file
- Linux: `.png` files in various sizes

## Icon Requirements

- Format: PNG with transparency
- Recommended size: 1024x1024 pixels
- Square aspect ratio
- Simple, recognizable design that works at small sizes

## Generating Icons

Once you have `photorec-refinery.png` in this directory, Briefcase will automatically
create all required icon formats during the build process.
